```bash

git clone https://github.com/vnodesign/gatsby-starter-lumen-dark && cd gatsby-starter-lumen-dark
yarn
yarn develop

```

To test the CMS locally, you'll need run a production build of the site:

```sh

yarn build
gatsby serve

```
